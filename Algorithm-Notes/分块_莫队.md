# 分块（含莫队）
## 一道例题引入
#### P3870 [TJOI2009] 开关
注意到暴力复杂度 $ O(n*m) $必然TLE，涉及区间修改，第一考虑线段树，可优化到$ O(m*logn) $,但此处我们介绍分块做法，时间复杂度$ O(m*\sqrt{n}) $
##### 何为分块？
 比如我们将$ m $个元素分成$n$块+最多$m$个元素，$O(m/n+m)$,根据均值不等式，当我们分为$ \sqrt{n}$块时，我们取得最小值。
##### 分析本题
题目要求处理一个初始全为 0 的灯泡序列，支持两种操作：
1.区间反转：将区间 $[L, R]$ 内的灯泡状态取反（0 变 1，1 变 0）
2.区间查询：查询区间 $[L, R]$ 内有多少个亮着的灯泡（状态为 1）。
##### 变量设置
  - `a[i]`：记录第 $i$ 个灯泡的基础状态（0 或 1）。
  - `belong[i]`：记录第 $i$ 个元素属于哪一个块。
  - `st[k]`, `ed[k]`：记录第 $k$ 个块的起始与结束下标。
  - `sum[k]`：记录第 $k$ 个块内亮着灯泡的数量（维护整块信息）。
  - `tag[k]`：记录第 $k$ 个块是否被打上了反转标记（Lazy Tag）。
##### 初始化
``` cpp
int n, m;
int a[100005];
int belong[100005], st[500], ed[500];
int sum[500], tag[500];
void build(){
    int block_size = sqrt(n);
    int block_num = n/block_size + (n%block_size != 0);
    //可能有不成块的元素
    for(int i=1;i<=block_num;i++){
        st[i] = (i-1)*block_size+1;
        ed[i] = min(i*block_size,n);
    }
    for(int i=1;i<=n;i++){
        belong[i] = (i-1)/block_size + 1;
        a[i] = 0;
    }
}
```
##### 区间操作
对于整块，直接异或tag[k]，sum更新为块长$-$原sum
对于离散的块，直接对 `tag[k]` 取反，并利用公式 `sum[k] = 块长 - sum[k]` 快速更新。
``` cpp
void change(int l,int r){
    int p = belong[l],q = belong[r];
    if(p==q){
        for(int i=l;i<=r;i++){
            if((a[i]^tag[p])==1) sum[p]--;
            else sum[p]++;
            a[i]^=1;
        }
    }else{
        for(int i=l;i<=ed[p];i++){
            if((a[i]^tag[p])==1) sum[p]--;
            else sum[p]++;
            a[i]^=1;
        }
        for(int i = st[q];i<=r;i++){
            if((a[i]^tag[q])==1) sum[q]--;
            else sum[q]++;
            a[i]^=1;
        }
        for(int k = p+1;k<=q-1;k++){
            tag[k]^=1;
            sum[k] = (ed[k]-st[k]+1) - sum[k];
        }
    }
}
```
##### 区间查询

``` cpp
int ask(int l,int r){
    int p = belong[l],q = belong[r];
    int ans = 0;
    if(p==q){
        for(int i=l;i<=r;i++){
            ans+=(a[i]^tag[p]);
        }
    }else{
        for(int i=l;i<=ed[p];i++){
            ans += (a[i]^tag[p]);
        }
        for(int i = st[q];i<=r;i++){
            ans += (a[i]^tag[q]);
        }
        for(int k=p+1;k<=q-1;k++){
            ans+=sum[k];
        }
    }return ans;
}
```
